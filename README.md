# 📚 CASHBOOK DASHBOARD

Aplicação de dashboard, com intuito de mostrar os movimentos de caixa de uma empresa.

### 📋 Como rodar a aplicação

Instale o composer através do comando 

```
composer install
```

Renomeio o arquivo env para .env <br>
Dentro do mesmo, ajuste o usuario e senha do banco de dados <br>
Coloque a pasta raiz da aplicação dentro do seu interpretador php <br>
Acesse o link local da aplicação <br>
